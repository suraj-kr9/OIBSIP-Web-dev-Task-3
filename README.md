# ✅ To-Do List Web App:

● A smart and interactive To-Do List application built with JavaScript, HTML, and CSS.

🚀 Features:

- Add and delete tasks
- Mark tasks as completed
- Filter tasks (All / Pending / Completed)
- Data persistence using LocalStorage

🛠️ Tech Stack:

- HTML
- CSS
- JavaScript

📚 What I Learned:

- State management in JavaScript
- LocalStorage usage
- Debugging real-world issues (index vs unique ID)

⚡ Future Improvements:

- Drag & drop tasks
- Task categories
- Cloud sync (Firebase)
